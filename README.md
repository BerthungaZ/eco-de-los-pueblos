# Eco de los Pueblos - roles, login y administración

Mejoras incluidas:
- Logo del navbar optimizado y más notorio.
- Pantalla de carga con logo.
- Animaciones suaves en secciones, tarjetas y botones.
- Apartado de iniciar sesión.
- Registro separado para cliente y artista.
- Rol administrador con acceso demo.
- Navbar dinámico por rol:
  - Cliente: compra y cuenta.
  - Artista: compra, cuenta y vender mi arte.
  - Administrador: panel administrador.
- Mi cuenta permite subir imagen de perfil.
- Mi cuenta permite guardar datos de envío.
- El carrito rellena automáticamente los datos de envío guardados.
- Panel administrador con métricas, usuarios, clientes, artistas, pedidos y botones de WhatsApp Web.
- Demo local con localStorage.
