Eco de los Pueblos - versión corregida para GitHub Pages

Esta versión está preparada para subirse a GitHub Pages con todos los archivos en la raíz.

Instrucciones:
1. En GitHub, elimina o reemplaza los archivos anteriores del repositorio.
2. Sube TODOS los archivos de esta carpeta directamente a la raíz del repositorio.
3. No subas la carpeta completa, sube el contenido de adentro.
4. GitHub Pages debe estar en: main / root.
5. Espera 1 minuto y abre la web.

- index.html
